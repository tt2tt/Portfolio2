# 概要

人生の半分は習慣的な行動からできている。しかし、多くの人が自分の習慣を正しく認識しておらず・新しい習慣の形成にも苦戦してしまう。 このアプリでは、科学的な観点から習慣作りをサポートし自分の意思で人生をコントロールしていくことを目指していく。
